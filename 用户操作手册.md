# 造价材料审计系统用户操作手册

## 1. 系统简介

造价材料审计系统是一个基于AI技术的智能审计平台，旨在帮助造价工程师和审计人员高效处理材料价格审核工作。系统通过建立市场信息价材料基准数据库，自动识别项目清单中的无信息价材料，运用AI技术分析价格合理性，并自动生成专业的审计报告。

**主要功能：**
*   **项目管理**：支持多项目并行管理，便捷导入项目材料清单。
*   **智能匹配**：自动匹配市场信息价，快速识别有价和无价材料。
*   **AI分析**：利用大模型技术对无信息价材料进行价格区间预测和风险评估。
*   **审计报告**：一键生成包含图表、数据分析和建议措施的Word审计报告。

## 3. 核心业务流程

完成一个项目的审计工作，通常需要遵循以下五个步骤：

### 第一步：创建项目
1.  进入左侧菜单 **“项目材料价格管理” -> “项目管理”**。
2.  点击右上角的 **“新建项目”** 按钮。
3.  填写项目基本信息（项目名称、编号、建设单位、施工单位等）。
4.  点击“保存”完成创建。

### 第二步：导入材料
1.  在项目列表中，找到刚创建的项目，点击 **“管理材料”** 或 **“导入”** 按钮。
2.  下载系统提供的 **Excel导入模板**。
3.  将您的项目材料清单按照模板格式整理好（确保包含材料名称、规格型号、单位、单价等关键列）。
4.  上传整理好的Excel文件，系统会自动解析并导入材料数据。

### 第三步：智能匹配
1.  材料导入后，系统会自动在后台运行匹配算法。
2.  系统会将项目材料与**市场信息价材料库**进行比对。
    *   **匹配成功**：自动关联基准价格，计算价格差异。
    *   **匹配失败**：标记为“无信息价材料”，等待AI分析。
3.  您可以在 **“项目材料详情”** 页面查看匹配结果，并手动调整匹配关系。

### 第四步：AI价格分析
1.  针对无信息价材料，系统支持AI智能分析。
2.  进入 **“价格分析” -> “分析结果”** 页面，选择对应的项目。
3.  点击 **“批量分析”**，系统将调用AI模型对未匹配的材料进行价格评估。
4.  分析完成后，您可以看到：
    *   **预测价格区间**（最低价、最高价、平均价）。
    *   **价格合理性评估**（正常、偏高、偏低）。
    *   **风险等级**（低风险、中风险、高风险）。
    *   **AI给出的专业建议和推理依据**。

### 第五步：生成报告
1.  所有分析确认无误后，进入 **“价格报告” -> “分析报告”**。
2.  点击 **“生成报告”**，选择需要生成报告的项目。
3.  系统会自动生成一份包含封面、目录、统计图表、详细数据对比和审计建议的Word文档。
4.  点击 **“下载”** 即可获取最终的审计报告。

## 4. 功能模块详解

### 4.1 项目材料价格管理
*   **项目列表**：查看所有审计项目，支持按名称、状态筛选。
*   **材料管理**：对项目内的材料进行增删改查，支持批量操作。
*   **上传记录**：查看材料导入的历史记录，方便追溯。

### 4.2 市场信息价材料
这里是系统的基准数据库，存储了各地的官方信息价或市场采集价。
*   **材料库**：查询和维护基准材料数据。
*   **Excel上传**：支持管理员批量导入最新的市场信息价数据。

### 4.3 无信息价材料
*   **无信息价材料库**：汇总所有项目中未匹配到信息价的材料。
*   **查询**：支持跨项目查询无价材料的历史分析记录。

### 4.4 价格分析
*   **分析结果**：核心分析界面，以图表和列表形式展示价格偏差。
*   **合理性判断**：系统根据AI预测价格和项目报审价格计算偏差率，自动判定风险等级。
    *   **正常**：偏差率 0%
    *   **低风险**：偏差率 ≤ 15%
    *   **中风险**：偏差率 15% - 30%
    *   **高风险**：偏差率 30% - 50%
    *   **严重风险**：偏差率 > 50%

### 4.5 业务规则说明
*   **不含税价格原则**：系统在进行价格对比分析时，统一使用**不含税价格**。请在导入数据时确保价格类型的准确性，系统会自动进行换算。
*   **匹配阈值**：
    *   **自动匹配**：相似度 > 85% 时，系统自动建立匹配关系。
    *   **人工确认**：相似度在 65% - 85% 之间时，建议人工复核。

### 4.6 系统管理
*   **用户管理**：管理员可以添加新用户、重置密码、配置角色权限。
*   **系统设置**：配置AI模型参数、匹配阈值等系统级选项。

## 5. 常见问题 (FAQ)

**Q1: 导入材料时提示格式错误怎么办？**
A: 请务必使用系统提供的Excel模板，不要修改表头名称。确保必填字段（如材料名称、单位）不为空。

**Q2: AI分析的结果准确吗？**
A: AI分析结果基于大数据模型预测，仅供参考。建议结合实际询价结果进行综合判断。对于高风险材料，建议人工复核。

**Q3: 如何修改匹配结果？**
A: 在项目材料详情页，点击某条材料的“匹配详情”，您可以手动搜索并选择正确的基准材料进行重新关联。

**Q4: 报告生成失败怎么办？**
A: 请检查项目是否包含材料数据，以及网络连接是否正常。如果问题持续，请联系系统管理员。

## 附录：术语表

*   **市场信息价材料**：在政府信息价库中有明确价格的材料。
*   **无信息价材料**：在政府信息价库中无对应价格的材料。
*   **基准数据库**：存储政府信息价的标准数据库，作为审计比对的基准。
*   **置信度**：AI预测结果的可信程度评分，分数越高表示AI对该预测越有把握。
