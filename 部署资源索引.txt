┌───────────────────────────────────────────────────────────────┐
│ 1. deploy-to-aliyun.sh - 一键部署脚本                        │
├───────────────────────────────────────────────────────────────┤
│ 重要程度: ⭐⭐⭐⭐⭐                                          │
│ 使用场景: 首次部署时使用                                      │
│ 执行时间: 15-20分钟                                           │
│ 使用方法:                                                     │
│   cd /opt/uma-audit5                                          │
│   ./deploy-to-aliyun.sh                                       │
│                                                               │
│ 脚本功能:                                                     │
│   ✓ 自动检查系统环境                                          │
│   ✓ 自动安装Docker和Docker Compose                           │
│   ✓ 生成安全的密码和密钥                                      │
│   ✓ 创建生产环境配置文件                                      │
│   ✓ 构建Docker镜像                                            │
│   ✓ 启动所有服务                                              │
│   ✓ 初始化数据库                                              │
│   ✓ 创建管理员账号 (可选)                                     │
│                                                               │
│ 注意事项:                                                     │
│   - 需要root权限执行                                          │
│   - 首次运行需要10-15分钟                                     │
│   - 记录显示的密码信息                                        │
│   - 部署完成后记得设置备份                                    │

1. 安全第一
   ✓ 所有密码必须妥善保管
   ✓ 定期修改密码
   ✓ 配置HTTPS保护数据传输
