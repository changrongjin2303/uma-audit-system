# 项目 21 综合导出包

## 文件夹结构

- `/reports/` - 审计报告
  - `audit_report.docx` - 完整审计报告

- `/raw_data/` - 原始数据
  - `project_info.json` - 项目基本信息
  - `materials.csv` - 材料清单数据

- `/analysis/` - 分析结果
  - `analysis_results.json` - 详细分析结果
  - `analysis_summary.csv` - 分析结果汇总

- `/attachments/` - 相关附件
  - 各种支撑文档和附件文件
